🤖 NagiBotV3 - WhatsApp MultiDevice Bot

Bienvenido a NagiBotV3, un poderoso bot para WhatsApp que funciona con Baileys MultiDevice, diseñado para ofrecer una experiencia completa, flexible y divertida en grupos y chats privados. Con herramientas modernas, funciones avanzadas y una interfaz amigable, NagiBotV3 se posiciona como uno de los bots más completos de su clase.

✨ Características Principales

Compatible con WhatsApp MultiDevice (última versión de Baileys)

Sistema modular de comandos (fácil de agregar o quitar funciones)

Comandos avanzados: IA, sticker, descarga, juegos, etc.

Herramientas para grupos: anti-link, anti-tox, protección spam

Configuración por grupo: bienvenidas, stickers, nombre, reglas, etc.

Bot multilenguaje (por defecto en Español)

Integración con IA: ChatGPT / Gemini (requiere API)

Sistema de perfiles, mascotas virtuales, niveles y economía

Interfaz visual: imágenes generadas por canvas

Soporte para botones, menús interactivos, respuestas personalizadas



---

📦 Requisitos

Node.js versión 18 o superior

Git

FFmpeg

Imagemagick (para stickers)

Terminal (Termux, consola Linux, CMD o PowerShell)



---

📥 Instalación

1. En Linux / Termux

pkg update && pkg upgrade
pkg install git nodejs ffmpeg imagemagick

# Clona el repositorio
git clone https://github.com/El-brayan502/NagiBotV3
cd NagiBotV3

# Instala dependencias
npm install

# Inicia el bot
npm start

> Aparecerá un código QR. Escanéalo con tu WhatsApp para emparejar el bot.




---

2. En Windows

Instala Node.js desde nodejs.org

Descarga Git: git-scm.com

Abre PowerShell y escribe:


git clone https://github.com/El-brayan502/NagiBotV3
cd NagiBotV3
npm install
npm start

🧠 Uso del bot

Una vez iniciado el bot, puedes usar los comandos con prefijo:

. por defecto (Ej: .menu, .play, .sticker)

Puedes cambiar el prefijo en la configuración


El bot responde tanto en chats privados como en grupos. Algunos comandos solo están disponibles para:

👑 Owner: Dueño del bot

🛡️ Admins: Administradores del grupo

👥 Todos: Cualquier usuario

🐾 Mascota Virtual

Sistema exclusivo estilo Tamagotchi:

.mascota – Muestra tu mascota

.alimentar – Alimenta a tu mascota

.jugar – Juega con tu mascota

.descansar – Hace que tu mascota duerma


Tu mascota evoluciona y reacciona según cómo la cuides 🐕


---

🧠 IA y Chatbot

Soporte para ChatGPT, Gemini u otros motores

Requiere ingresar una API KEY en la configuración

Comando: .ai Hola, ¿cómo estás?

Opcional: activar IA automática en grupo

📥 Descargadores
